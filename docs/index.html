About

Russian Roulette is a Text User Interface (TUI) game that simulates the classic Russian Roulette mechanic in a completely safe, non-destructive way.

Built with Rust, ratatui, and real-time input handling, the game focuses on suspense, minimalism, and cross-platform support.

Features

  • Terminal UI (TUI)
  • Real-time key input
  • Six-chamber revolver simulation
  • Suspense-based trigger delay
  • Victory after six safe rounds
  • Safe exit anytime

Platforms

  • Linux (x86_64)
  • Windows (x86_64)
  • macOS (ARM64)
  • Android (Termux, aarch64)
